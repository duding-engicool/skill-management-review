# ISO 9001:2015 第 9.3 条 管理评审（内置知识）

> 条款要点与输入映射，供生成评审报告时引用。IATF 16949 补充要求以「待企业补充」标注。

## 一、9.3.1 总则

最高管理者应按策划的时间间隔（通常每年至少一次）评审组织的质量管理体系，以确保其持续的适宜性、充分性和有效性，并与组织的战略方向一致。

## 二、9.3.2 管理评审输入

应基于以下方面作为输入信息：

| 子项 | 输入内容 | 本技能映射 |
|------|----------|------------|
| a) | 以往管理评审所采取措施的状况 | 以往评审跟踪措施 |
| b) | 与质量管理体系相关的内外部因素的变化 | 体系变更 |
| c) | 有关质量管理体系绩效和有效性信息，包括趋势：①顾客满意和相关方反馈 ②质量目标的实现程度 ③过程绩效及产品和服务的符合性 ④不合格及纠正措施 ⑤监视和测量结果 ⑥审核结果 ⑦供方 | 顾客反馈 / 质量目标 / 过程绩效 / 纠正措施 / 审核结果 |
| d) | 资源的充分性 | 资源需求（输出） |
| e) | 改进的机会 | 改进建议 |

> IATF 16949:2016 在 9.3.1.1（补充）中扩展了输入，如不良质量成本、保修、现场失效、交付中断、顾客计分卡等，「待企业补充」。

## 三、9.3.3 管理评审输出

管理评审的输出应包括与以下有关的决定和措施：

- 改进的机会
- 质量管理体系所需的变更
- 资源需求

输出应保留成文信息（记录）。

## 四、有效性评价三维

- **适宜性**：体系是否仍适合组织的环境、战略与目的。
- **充分性**：资源、过程、职责是否足够支撑目标实现。
- **有效性**：体系是否实现预期结果（质量目标、顾客满意等）。

定性结论建议：充分有效 / 基本有效需局部优化 / 不充分需重大整改。

> 趋势统一表述：上升 / 稳定 / 下降。
